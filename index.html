Using the Analytics

Please use the following: <script src=//analytics.global.sky.com/{{ site.version }}/analytics.js></script> (this works over http or https). If your project uses RequireJS, Click here to see a requireJS example.

Then add the following onto your page with the correct account details to initialise the tracking.

analytics.trackPage({ site: 'global', section: 'skyglobal/analytics', account: 'bskybdemodev', page: 'Analytics demo page' });

Boom. Job Done.

Now anchors, buttons, submits and any element with [data-tracking] are automatically tracking.
