Custom Variables

You will need variable Name and Omniture ID. These should be added to the tracking config on initialisation.

var config = { ... customVariables: [ {'myCustomVar': {var: 71}} ] }; toolkit.tracking.pageView(config);

To get a value to go with your variable, you must bind it to an HTML element using the attribute data-tracking-variable and setting it to your variable name. This element also needs the attribute data-tracking-value so we know what to send to Omniture. If this attribute is omitted, the value of the item clicked will be tracked.

Examples

Tracking drink and setting it to a text box



<label for='input_field'>What is your favourite drink?</label> <input id='input_field' name='input_field' type='text' /> <button data-tracking-variable='drink' data-tracking-value="" class='btn primary-cta'>Submit</button>

Tracking how_about_pina_coladas and setting it to the selected radio button



<label>Do you like Pina Coldas?</label> <input id='radio_yes' name='radio_field' type='radio' value='yes_to_pinas'/><label for='radio_yes'>Yes</label> <input id='radio_no' name='radio_field' type='radio' value='no_to_pinas'/><label for='radio_no'>No</label> <button data-tracking-variable='how_about_pina_coladas' class='btn primary-cta'>Submit</button>
<button id='colour-button-blue' data-tracking-variable="colour">Blue</button>
