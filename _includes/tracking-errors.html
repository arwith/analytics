Error Tracking

Tracking an error is as easy 1, 2, 404. You can either send an error on pageView or via JS.

Examples

Send Error
Manually call the analytics.trackError function.
HTML
none
Javascript
analytics.trackError('myCustomError i.e. 404 or no-package-found');
Send Error on pageView
All you need to do is use loadVariablesin the pageView config.
HTML
none
Javascript
SITECAT_CONFIG = { site: 'global', section: 'skyglobal/analytics/debug', headline: '', contentType: 'demo', contentId: '', account: 'bskybglobaldev', page: 'Analytics debug demo page', loadVariables: { errors: '404' } }; analytics.pageView(SITECAT_CONFIG);
